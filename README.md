# sample-vuetify2

Trying out the vue project scaffolding with vuetify plugin.

- create regular vue cli plugin
- install cli plugin with `npm i -D vue-cli-plugin-vuetify`
- run `vue invoke vue-cli-plugin-vuetify`

## Project setup

```
npm install
```
